# gin-blog

## 目录结构

```
- gin-blog
    - conf - 配置文件
    - controller - 控制器
    - dao - 数据库连接
    - log - 日志文件
    - logger - 日志记录器（zap日志工具类）
    - models - GORM访问数据库
    - note - 开发笔记
    - routers - 路由
    - settings - 文件配置管理
    - static - 前端静态文件
    - templates - 页面文件
    - util - 工具类
    - go.mod - Go模块管理文件
    - main.go - Go程序入口文件
    - README.md - 项目介绍文件
```
